Client e Servers:

Sobre os Client Sabemos Que são os que vão Buscar a informação Dentro Dos Servidores. E o Que são Servidores?(Servers), São Os que Leva as informações, O que entrega A informações para os Clientes, no proprio nome ja fala , a Função do Servers.

OBS:(Client São os Que Vão Buscar as informações, Atraves Da Navegação Web, usando o Protocolo http)

Conceitos Sobre Client

  • Diferentes resoluções, deve ser considerado na criação de websites

  • Também a compatibilidade entre navegadores (Browsers)

Sobre Navegadores

  • São programas criadas por empresas, utilizados para abrir/executar arquivos.

  • Seguem padrões W3C, porém, sempre tem uma diferença ou outra de interpretação.

  • Também chamados de “browsers” iniciou com o MOSAIC, passou para o Netscape, e hoje temos uma variedade de navegadores disponíveis.

  • Gratuitos

Aplicações Web

Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação.

Conceitos Dos Servers

Sabendo Que os Clientes Vão Buscar As informações, Dentro do Servers São armazenado: Arquivo, Segurança (Firewall) , Streaming, E-hmail , web . Algumas Empresas Tem um Computador que literalmente É a cabeça Da empresa , Onde esta Armazenado Tudo isso Que foi Citado. Ondem, podem Tambem conter Uma Cabine Com tudo isso Dentro Guardado, Como Se fosse um Gabinete Gigante.
